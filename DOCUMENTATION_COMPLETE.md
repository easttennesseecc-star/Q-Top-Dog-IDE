## Glossary

### Key Terms Used

**OAuth 2.0**: Authorization framework used for sign-in
**Code Exchange**: Trading authorization code for access token
**JWT Token**: JSON Web Token used for authentication
**Scopes**: Permissions requested during OAuth
**Revocation**: Removing stored credentials
**CSRF**: Cross-Site Request Forgery attack (prevented by state parameter)
**Credentials**: OAuth tokens or API keys
**Pre-use Validation**: Checking auth before using LLM
