## ğŸš¨ Troubleshooting

### Problem: Launcher stops on a step

**Solutions (in order):**
1. Wait 5 seconds - it might just be slow
2. Read the error message shown
3. Check relevant section in `FLAWLESS_STARTUP_GUIDE.md`
4. Run `TEST_LAUNCHER_CONFIG.bat` to diagnose
5. Close all windows and try again

### Problem: "Python not found"
- Install Python 3.11+ from https://python.org
- **IMPORTANT:** Check "Add Python to PATH"
- Restart computer
- Try launcher again

### Problem: "pnpm not found"
- Run in PowerShell: `npm install -g pnpm`
- Try launcher again

### Problem: "Port in use" error
- Close other apps using ports 8000 or 1431
- Try launcher again
- Or: Wait 30 seconds and try again
