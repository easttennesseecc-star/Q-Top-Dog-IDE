## ğŸš¨ Troubleshooting

### Issue: "Python not found"
```
Solution:
1. Install Python 3.11+ from python.org
2. CHECK "Add Python to PATH" during installation
3. Restart your computer
4. Try launcher again
```

### Issue: "pnpm not found"
```
Solution:
1. Open PowerShell
2. Run: npm install -g pnpm
3. Try launcher again
```

### Issue: "Backend won't start"
```
Solution:
1. Check if port 8000 is in use
2. Run: netstat -ano | findstr :8000
3. If found, kill the process
4. Try launcher again
```

### Issue: "Frontend won't start"
```
Solution:
1. Check if port 1431 is in use
2. Run: netstat -ano | findstr :1431
3. If found, kill the process
4. Try launcher again
```

### Issue: "Launcher hangs"
```
Solution:
1. Press Ctrl+C to cancel
2. Kill all Python: taskkill /F /IM python.exe
3. Kill all Node: taskkill /F /IM node.exe
4. Try launcher again
```
